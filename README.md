# LibraryDB-PostgreSQL-Analysis
A project focused on database efficiency, query planning, and security constraints using PostgreSQL for The University of Queensland.
LibraryDB: Database Security & Query Optimization
This project demonstrates advanced PostgreSQL techniques focusing on data integrity and performance analysis.

* Cyber Defense: Implemented CHECK constraints and Triggers to enforce business logic and data security. * Data Analytics: Created complex Views and Common Table Expressions (CTEs) to analyze patron behavior and charges. * Performance Tuning: Utilized EXPLAIN ANALYZE to interpret query plans and optimized retrieval times using B-Tree and Function-based indexes. * Concurrency: Managed Transactions to ensure data consistency during simultaneous user actions.
